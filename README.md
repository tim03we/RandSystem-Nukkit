# RandSystem (Nukkit)

This is an edge system for the plugin PlotSquared on Nukkit. With this plugin you can change the edges of your plot.

# Commands
Command | SubCommand | Description | Permission 
--------- | ------- | ------------------- | ------- 
/p | rand | Change your plot border | plots.rand
